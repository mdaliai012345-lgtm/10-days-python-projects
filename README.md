# Jarvis Voice Assistant

This is a Python project that opens websites like Google, LinkedIn, Facebook, and YouTube using voice commands.

I recently completed my Python course and built this project by following the CodeWithHarry tutorials and adding my own modifications.

## Features
- Open Google
- Open Facebook
- Open YouTube
- Open LinkedIn
- Voice recognition
- Text-to-speech responses

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- webbrowser
